# Security Policy

## Reporting a vulnerability

If you find a security issue in chime, please **do not** open a public GitHub issue.

Instead, email <sarthakmahapatra.dev@gmail.com> with:

- a description of the issue,
- steps to reproduce,
- the chime version and OS you're running.

I'll respond within a week and coordinate a fix and disclosure timeline.

## Supported versions

Only the latest minor release on PyPI receives security fixes.
